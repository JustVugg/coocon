# Security Policy

## Supported Versions

Only the latest commit on `main` is supported for security fixes.

## Reporting a Vulnerability

Please report security issues privately by email:

- security@coocon.dev

Include:
- A clear description of the issue
- Steps to reproduce
- Impact assessment if known

We will respond within 72 hours.

## Scope

Coocon is a secure execution platform. However, it is **not** a VM-grade isolation boundary.
If you require strong multi-tenant isolation, combine Coocon with OS/container/microVM isolation.
